Help & Tips
Coupon Code Guidelines:
  • Use uppercase letters and numbers
  • Keep it short and memorable
  • Avoid special characters
  • Make it relevant to the discount
Editing Considerations:
  • Changing the code affects existing shares
  • Reducing usage limits may affect availability
  • Consider customer communication for major changes
Best Practices:
  • Monitor usage statistics regularly
  • Archive unused coupons periodically
  • Communicate changes to customers
