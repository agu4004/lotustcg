CSV Format

Your CSV file should contain the following columns:

Column Required Description Example
name Yes Card name Lightning Bolt
set_name No Set or expansion name Core Set
rarity No Card rarity Common, Rare, Super Rare, Majestic, Legendary, Promo, Treasure, Marvel
condition No Card condition Near Mint, Light Play, Moderately Play
language No Card language English, Not English
price No Price in dollars 1.50
quantity No Number of copies 10
description No Card description Classic red instant spell
image_url No Direct link to card image https://example.com/card.jpg
foiling No Card foiling type NF, RF, CF
art_style No Card art style Normal, EA, WB, ALT, GF, Marvel
card_code No Optional external code/identifier for the card BL-ALPHA-000
